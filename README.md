## Getting Started
### Prerequisites
- pnpm 8+
- Node.js 18+
- Astro 5.12.3

### Installation
1. Clone the repository with the following commands
```bash
git clone https://github.com/borjaherranz/portfolio
```

2. Install dependencies
```bash
pnpm create astro@latest
pnpm astro add tailwind
```
